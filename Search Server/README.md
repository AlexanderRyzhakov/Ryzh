# SearchServer
---
Ядро поисковика документов с учетом минус- и стоп-слов. Вывод результатов с ранжированием по TF-IDF. Используются асинхронные вычисления и юнит-тестирование.
 
## Возможности: 
---
1.	Добавление и удаление документа
2.	Поиск документов по запросу, документы, содержащие минус-слова не выводятся
3.	Возможна разбивка результатов поиска на группы для последующего постраничного вывода.

## Системные требования:
---
1.	С++17
2.	GCC 11.20
